## CMIN2 到底是什么线路，和普通 CMI 有什么区别

一句话版本：CMIN2 是中国移动自己搭建的高端国际骨干网，ASN 号 AS58807，定位跟电信 CN2 GIA、联通 CUP 一个级别。

普通 CMI 用的是 AS58453，在 2022 年前后明显感觉越来越堵。CMIN2 是移动后来专门为高质量出海流量另开的通道，跟国内移动骨干网（AS9808）直连，绕开了大量公共交换节点。实测延迟低、抖动小。你可以简单理解成：移动版的 CN2 GIA。

搬瓦工洛杉矶移动线路走 CMIN2 之后，移动用户的回程路由就从"绕来绕去的普通 CMI"变成了"国内入境一跳入、一跳落地"这种更干净的路径。
